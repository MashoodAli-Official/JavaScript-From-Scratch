Scope in JavaScript

The scope of a variable is the region in which that variable is accessible.

  • Global Scope: The variable is declared outside the function is called global scope.
  • Local Scope: The variable is declared inside the function is called local scope.
  • Block Scope: Before ES6 the only two type of scope were introduces, Global scope and local scope. In ES6 the new variable keywords (let and const) were introduced to declare new variables. Any variable declared with let or const inside {} block, it will not access outside the block.
