Map Methods in Javascript

Some of the most commonly used Map methods are:
  • new Map(): method you can cerate a new map by passing an array to the new Map() constructor
  • get(): By this you can get the value of the key of the map.
  • set(): By this you can add/set the values to map. By set() method you can also change the existing map values
  • size(): By this you can get the size of the map.
  • delete(): By this method you can delete the element of the Map.
  • has(): By this method you can check the key is present in the map or not.
