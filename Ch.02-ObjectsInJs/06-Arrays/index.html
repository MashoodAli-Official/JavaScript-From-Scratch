Arrays In JavaScript

An array is a special variable, which can hold more than one value. It is a zero-indexed collection of values. It is used to store multiple values in a single variable.

Basic Syntax

const arrayName = new Array(value1, value2, value3, ...); const arrayName = [value1, value2, value3, ...];

Array methods
  • arrayName.sort()

    • The sort() method sorts an array in ascending order.

  • arrayName.reverse()

    • The reverse() method reverses the order of the elements in an array.

  • arrayName.join()

    • The join() method joins all elements of an array into a string.

  • arrayName.concat()

    • The concat() method creates a new array by merging (concatenating) existing arrays.

  • arrayName.pop()

    • The pop() method removes the last element from an array and returns that element.

  • arrayName.push()

    • The push() method adds one or more elements to the end of an array and returns the new length of the array.

  • arrayName.shift()

    • The shift() method removes the first element from an array and returns that element.

  • arrayName.unshift()

    • The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.

  • arrayName.slice()

    • The slice() method returns a shallow copy of a portion of an array into a new array object.

  • arrayName.splice()

    • The splice() method changes the contents of an array by removing existing elements and/or adding new elements.

  • arrayName.length

    • The length property returns the number of elements in an array.

  • arrayName.map()

    • The map() method creates a new array with the results of calling a provided function on every element in the calling array.

  • arrayName.flat()

    • The flat() method creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.
