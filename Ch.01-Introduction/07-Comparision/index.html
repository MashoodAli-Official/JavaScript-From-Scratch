Comparison in Javascript

There are different types of comparison operators which are essentially use in Javascript.

  • (==) Equal to
  • (===) Equal to check both values and type of variables.
  • (!=) Not Equal
  • (>) Greater than
  • (<) Less than
  • (>=) Greater than equal
  • (<=) Less than equal
  • (!==>) Not equal, check both values and types of values
